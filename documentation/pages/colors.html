Colors & Theming

ArarA ships with a comprehensive, curated color system. Every color generates text-, bg-, border-, and hover variants — giving you thousands of utility classes from a single, consistent palette.

Semantic Aliases

Use these semantic names to express intent rather than specific hues. They map to the corresponding color in the palette and can be overridden in your Sass configuration.

primary

#3b82f6

#ef4444

HTML
<p class="text-primary">Primary text</p>
<div class="bg-danger text-white p-4">Danger alert</div>
<span class="bg-success/20 text-success px-2 py-1 rounded">Active</span>

Text Color

Use text-{color}-{shade} to set text color.

Indigo 600 - Primary Focus

Slate 900 - Strong Heading

Slate 600 - Standard body text for legibility.

Success Green - Confirmed state.

HTML
<p class="text-indigo-600 font-bold">…</p>
<p class="text-slate-900 font-semibold">…</p>
<p class="text-success font-medium">…</p>

Background Color

Use bg-{color}-{shade} to apply a background. Pair with opacity modifiers for glassmorphism effects.

bg-indigo-600

bg-slate-900

bg-green-100

bg-red-50

bg-amber-400

bg-indigo-500/20

HTML
<div class="bg-indigo-600 text-white p-4 rounded-xl">…</div>
<div class="bg-green-100 text-green-800 p-4 rounded-xl">…</div>
<!-- Opacity modifier -->
<div class="bg-indigo-500/20 text-indigo-700">…</div>

Hover State Variants

Prefix any color utility with hover: to apply it on mouse hover.

HTML
<button class="bg-indigo-600 hover:bg-indigo-700 text-white">…</button>
<a class="text-slate-600 hover:text-indigo-600">…</a>
<div class="border-slate-300 hover:border-indigo-400">…</div>
