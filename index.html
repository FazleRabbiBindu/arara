Introduction

A utility-first CSS library designed for speed and flexibility. Built with Sass, optimized for the modern web with 2026 standards in mind.

Installation

Include the compiled CSS file in your project:

<link rel="stylesheet" href="dist/css/style.css">

Responsive Design

Every utility can be applied conditionally at different breakpoints using our responsive prefixes.

Available Breakpoints

  • sm: 640px
  • md: 768px
  • lg: 1024px
  • xl: 1280px

Usage Example

class="w-full md:w-1/2 lg:w-1/3"

Adjusts width based on screen size.
