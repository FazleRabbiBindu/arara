Text truncation

Line clamping

Limit text to a specific number of lines.

<p class="line-clamp-2">Long paragraph that will be clamped to 2 lines...</p>

This is a long paragraph that demonstrates line clamping. It will be limited to exactly two lines of text, with any overflow hidden and an ellipsis added at the end. This is useful for card summaries or preview text where you want to maintain consistent heights.

Border radius sizes

Choose a radius utility to control how rounded the corners are.

  • .rounded-none square corners
  • .rounded-sm | small radius
  • .rounded | default radius
  • .rounded-md | medium radius
  • .rounded-lg | large radius
  • .rounded-xl | extra large radius
  • .rounded-2xl / .rounded-3xl | extra-curved corners
  • .rounded-full | pill / full circle corners

Border radius corners

Apply radius to specific corners or edges only.

  • .rounded-tl, .rounded-tr, .rounded-br, .rounded-bl | single corners
  • .rounded-t, .rounded-r, .rounded-b, .rounded-l | top, right, bottom, left edges
  • Each corner/edge also supports size suffixes like .rounded-tl-lg or .rounded-b-sm

Grid

StyleLib grid utilities let you build responsive layouts with explicit column, row, span, and flow control.

Grid utilities for structure
Responsive grid classes
Preview-ready examples

Basic setup

Start with .grid or .inline-grid, then choose column and row templates.

<div class="grid grid-cols-3 gap-4">
  <div class="col-span-2">Wide item</div>
  <div>Box</div>
</div>

Column templates

Use:

  • .grid-cols-1 through .grid-cols-12
  • .grid-cols-none
  • .grid-cols-subgrid

Row templates

Use row templates the same way:

  • .grid-rows-1 through .grid-rows-12
  • .grid-rows-none
  • .grid-rows-subgrid

Span utilities

Span items across multiple tracks:

.col-span-2
1
.row-span-2
2
<div class="col-span-2">...</div>
<div class="row-span-2">...</div>

Line placement

Position items with start/end classes:

  • .col-start-1 through .col-start-13
  • .col-end-1 through .col-end-13
  • .row-start-1 through .row-start-13
  • .row-end-1 through .row-end-13

Negative lines are available with .-col-start-#, .-col-end-#, .-row-start-#, and .-row-end-#.

Shorthand placement

Use direct track placement classes:

  • .col-auto, .row-auto
  • .col-1 through .col-12
  • .row-1 through .row-12

Auto flow

Control placement direction and packing:

  • .grid-flow-row
  • .grid-flow-col
  • .grid-flow-dense
  • .grid-flow-row-dense
  • .grid-flow-col-dense

Auto tracks

Use these classes for implicit grid sizing:

  • .auto-cols-auto, .auto-cols-min, .auto-cols-max, .auto-cols-fr
  • .auto-rows-auto, .auto-rows-min, .auto-rows-max, .auto-rows-fr

Responsive utilities

Prefix any grid utility with breakpoint names like .md:grid-cols-6 or .lg:col-span-4.

<div class="grid grid-cols-1 md:grid-cols-3 gap-4">
  <div class="col-span-1 md:col-span-2">Responsive item</div>
  <div>Item</div>
</div>
