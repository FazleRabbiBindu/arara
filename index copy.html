Getting Started

StyleLib is a lightweight utility-first CSS framework with reusable components, grid support, flex utilities, spacing helpers, and responsive design classes.

npm install @bindu/style-lib

Usage Manual

Combine utility classes with component blocks to build layouts and UI quickly. Use grid or flex for layout, spacing utilities for margins and padding, and color classes for backgrounds and text.

  • Start with layout: use .row, .grid, or .flex.
  • Spacing: use .p-4, .m-3, .px-2, and responsive variants like .md:p-4.
  • Color: use background utilities like .bg-primary and text utilities like .text-white.
  • Components: use .btn, .card, .table, and .form-control.
<div class="row gap-4">
  <div class="col-12 md:col-6">...</div>
  <div class="col-12 md:col-6">...</div>
</div>
